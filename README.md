# Nexus AI SaaS

AI-powered SaaS platform landing page built with HTML, CSS & JavaScript.

## Features
- Live AI demo chat (powered by Groq / Llama 3)
- Sign up & Sign in (localStorage auth)
- Full dashboard with API key management
- Models, Playground, Analytics, Settings sections
- Dark/Light theme toggle
- Fully responsive

## Pages
- `index.html` — Landing page + live demo
- `signup.html` — Account creation
- `signin.html` — Sign in
- `dashboard.html` — User dashboard
- `verify-email.html` — Email verification
